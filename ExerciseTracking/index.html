Exercise Tracking with OOP

This project demonstrates inheritance, encapsulation, and polymorphism in C#, using three activities: Running, Cycling, and Swimming.

About the project

  • Base: Activity (date and minutes).
  • Derived: Running, Cycling, Swimming.
  • Abstract methods: GetDistanceKm, GetSpeedKph, GetPaceMinPerKm.
  • GetSummary() in the base class, calling polymorphic methods.

How to run

  1. Create the project: 
    dotnet new console -n ExerciseTracking
cd ExerciseTracking
  2. Add Activity.cs, Running.cs, Cycling.cs, Swimming.cs, and replace Program.cs with the provided one.
  3. Run: 
    dotnet run

Repository: github.com/caioprosa10/ExerciseTracking

Rubric — fully met ✅

  • Abstraction (base + derived)
  • Encapsulation (private fields)
  • Inheritance (common attributes in base)
  • Override (distance/speed/pace)
  • GetSummary() in base (virtual)
  • List with all 3 activities
  • No UI/user interaction
  • Correct naming and file organization
